Name
====

Greeting

Description
===========

A Finagle/Finch microservice that renders a simple JSON greeting.

Building
========

To build, run the following SBT command:

    $ sbt dist

Running
=======

To run:

    $ java -jar target/scala-2.11/greeting-0.1.jar

To run with sbt:

    $ sbt run
